Minimalist reactive data binding for vanilla JavaScript

What is rnxJS?

rnxJS is a lightweight reactive data binding library that brings the power of reactive programming to vanilla JavaScript. Built with simplicity in mind, it provides a clean API for creating reactive state that automatically updates your UI when data changes.

Just 16kb bundled - no bloat, no dependencies

Automatic UI updates when state changes

Easy to learn, powerful to use

import { createReactiveState } from '@arnelirobles/rnxjs';

const state = createReactiveState({ count: 0 });

// Subscribe to changes
state.subscribe('count', (value) => {
  document.getElementById('counter-value').textContent = value;
});

// Update state - UI updates automatically
document.getElementById('increment-btn').onclick = () => state.count++;
document.getElementById('decrement-btn').onclick = () => state.count--;
document.getElementById('reset-btn').onclick = () => state.count = 0;

const state = createReactiveState({ name: '' });

// Input updates state
document.getElementById('name-input').addEventListener('input', (e) => {
  state.name = e.target.value;
});

// State updates output
state.subscribe('name', (value) => {
  const display = value || 'Guest';
  document.getElementById('name-output').textContent = display;
  document.getElementById('char-count').textContent = value.length;
});

  if (input.value.trim()) {
